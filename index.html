üìö Theory

Biometric authentication uses unique physical characteristics for identity verification. Combined with cryptographic hashing, biometric data is transformed into secure, irreversible hash values.

Key concepts: SHA-256 hashing, salt generation, one-way functions, hash collision resistance, and secure storage principles.

Enrollment Phase:

1. Capture biometric data (fingerprint/face)
2. Generate random salt
3. Compute SHA-256 hash(biometric + salt)
4. Store hash and salt securely

Verification Phase:

1. Capture new biometric sample
2. Retrieve stored salt
3. Compute hash with same salt
4. Compare with stored hash
